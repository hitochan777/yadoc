Yet Another Docker CLI

## Requirements
- Python >= 3.5
- pipenv
## Install

```
git clone https://github.com/hitochan777/yadoc.git
cd yadoc
pipenv install
```

## Features

### List tags

Example
```
$ ppr python main.py tag weseek/growi --limit 5
3.1.7
3.1.6
3.1.5
3.1.2
3.1.1
```
